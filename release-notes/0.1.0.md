# slidex 0.1.0

## Highlights

- Packages the slidex CLI with deck templates, schemas, Codex plugin files, repository marketplace metadata, and install guidance for local use.
- Provides GitHub Actions release packaging for Linux, macOS, and Windows on amd64 and arm64.
- Includes SHA-256 checksum generation and GitHub artifact attestation generation for release assets.

## Added

- Initial release package workflow for production and canary channels.
- Local update metadata for release-package installs.
- Repository health checks for version metadata, plugin metadata, protocol bundle, and update state.
